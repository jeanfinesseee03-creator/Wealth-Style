# Wealth Style

Wealth Style is a modern clothing brand website designed to showcase premium streetwear and luxury-inspired fashion.

- Modern space-themed design
- Home page
- Shop page
- Cart page
- Contact page
- Mobile-friendly layout
- Animated star background
